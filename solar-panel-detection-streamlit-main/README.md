# Solar Panel Detection (Streamlit)

Streamlit app for solar panel detection using Roboflow and optional Google Maps satellite imagery.

## Quick run

1. **Install:** `pip install -r requirements.txt`
2. **Configure:** Copy `env.example` to `.env` and set `ROBOFLOW_API_KEY` (and optionally `GOOGLE_API_KEY` for lat/long).
3. **Run:** `streamlit run streamlit_roboflow_7.py`

See **[RUN.md](RUN.md)** for full setup, run commands, and API notes.
